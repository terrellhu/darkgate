# 孢衣 (Spore Veil) — 瘟疫使者 · char_plague_01

> ID: `char_plague_01` | 职业: PLAGUE（瘟疫使者） | 稀有度: R | **异化系职业**

## 角色概要

第三阶段感染者，全身被真菌孢子覆盖。孢子在她的控制下可定向感染，但她本人已对毒素完全免疫。说话时伴随轻微咳嗽声，自称"我们"。

## 视觉设定

- **性别：** 女性（女武神 / 异化感染者）
- **体型：** 纤瘦病态，骨架明显，皮肤苍白近乎半透明
- **发型：** 灰绿色长发凌乱披散，发丝间生长着微小的荧光真菌菌丝
- **面部：** 病态美感，眼窝深陷，虹膜变异为荧光黄绿色，瞳孔为竖瞳（非人类），嘴角和鼻周有真菌孢子粉尘
- **服装：** 破旧的生化防护服（已被真菌从内部侵蚀穿透），防护面罩挂在脖子上已失效，胸口生物危害标志
- **异化特征：** 双手和前臂表面覆盖暗绿色真菌菌落，指尖有不断释放的微小孢子粒子（荧光黄绿），右肩胛骨处长出蘑菇状突起，皮肤纹理有菌丝网络
- **武器：** 不持传统武器——双手向前伸展时释放大量毒性孢子云
- **色彩重点：** 灰白病态肤色+暗绿真菌+荧光黄绿孢子光效，生化防护服的暗黄旧色
- **气质：** 令人不安的病态优雅，表情平静甚至带有诡异微笑，既是受害者也是行走的瘟疫
