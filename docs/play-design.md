# 玩法设计文档：《异化防线：九重黑门》

本文档详细描述游戏的各项玩法机制和系统规则，作为开发实现的直接参考。

---

## 1. 核心循环

游戏由两个互相依存的循环驱动：

- **内循环（枢纽经营）**：生产资源 → 维系幸存者生存 → 制造装备/强化角色
- **外循环（黑门探索）**：消耗资源外出 → 获取稀有素材/推进剧情 → 反哺基地

玩家在两个循环间不断切换，形成"经营-探索-经营"的节奏。

## 2. 资源体系

### 2.1 基础资源

| 资源名 | 图标色 | 用途 | 获取方式 |
| --- | --- | --- | --- |
| 生物电 (Bio-Electricity) | 青色 | 所有幸存者的生命线，探索行动力 | 维生反应堆产出 |
| 纳米合金 | 银色 | 升级设施、锻造装备 | 探索拾取、拆解装备 |
| 废弃芯片 | 橙色 | 升级设施、角色义体改造 | 探索拾取、回收死亡队员 |
| 算力 (Hashrate) | 紫色 | 招募、训练、解密档案（通用货币） | 探索奖励、任务完成 |
| 信用点 | 金色 | 黑市交易专用 | 稀有掉落、出售物品 |

### 2.2 特殊资源

| 资源名 | 用途 |
| --- | --- |
| 主角精神力 | 开启理智广播、探索时维持精神链接，有上限且缓慢恢复 |
| 角色异化值 | 异格者专属，释放强力技能累积，到临界点失控 |
| SAN值（理智） | 探索时全队共享，降至0则队伍崩溃强制撤退 |

## 3. 枢纽经营系统

### 3.1 设施列表

| 设施 | 对应地堡原型 | 功能 | 升级效果 |
| --- | --- | --- | --- |
| 维生反应堆 | 庄园 | 分配工人产生生物电 | 提高产出效率、容量上限 |
| 神经接入舱 | 酒馆 | 广播招募新角色 | 解锁更高稀有度池、降低消耗 |
| 义体诊所 | 教堂/墓地 | 治疗伤员、回收死亡队员芯片 | 提高治疗速度、芯片继承率 |
| 黑市终端 | 市场 | 随机刷新商品，信用点交易 | 增加刷新槽位、降低价格 |
| 纳米锻造间 | 铁匠铺 | 用蓝图+材料锻造装备 | 解锁高级蓝图、增加词条槽 |
| 数据解析室 | 图书馆 | 解密收集到的数据终端和音频日志 | 加速解密、解锁隐藏剧情 |

### 3.2 理智广播（核心创新）

- 主角消耗精神力开启"理智广播"
- 广播范围越广 → 招募到高品质角色（S/SS级）的概率越高
- 广播结束后主角进入"精神疲劳"状态 → 基地生产效率临时下降
- 玩家需要权衡：立刻招募 vs 维持生产

### 3.3 工人分配

- 幸存者分为"工人"和"战斗员"两类
- 工人分配到各设施增加产出
- 战斗员编入小队外出探索
- 部分角色可在两种角色间切换（效率有损耗）

## 4. 探索系统

### 4.1 基础机制

- **地图结构**：9扇黑门 = 9个大章节地图，每个地图由若干节点组成网格
- **移动方式**：走格子，每步消耗生物电
- **迷雾机制**：未探索区域被迷雾覆盖，逐步揭开
- **精神链接值**：替代传统"火把/光照"
  - 高链接值：视野广，暴击率提升，可提前发现敌人弱点
  - 低链接值：屏幕出现噪点干扰，遭偷袭概率大增，队员出现幻觉（自动攻击队友/拒绝行动）

### 4.2 节点类型

| 节点类型 | 图标 | 说明 |
| --- | --- | --- |
| 空地 | 灰色圆点 | 无事件，安全通过 |
| 战斗 | 红色骷髅 | 遭遇敌人，进入战斗 |
| 事件 | 蓝色问号 | 触发随机/固定文字事件，可能有选择分支 |
| 宝箱 | 金色箱子 | 获得物资/装备 |
| 数据终端 | 绿色屏幕 | 收集碎片化叙事文本 |
| 补给站 | 白色十字 | 恢复部分生物电和SAN值 |
| BOSS | 暗红门扉 | 章节BOSS战 |

### 4.3 Roguelike 支线

- 主线地图固定，支线区域随机生成
- 支线包含随机事件、陷阱、隐藏宝箱
- 每次进入支线区域布局不同，增加重复可玩性

### 4.4 SAN值（理智）详细规则

- 探索黑门时全队共享一个SAN值，随时间和事件持续下降
- 主角消耗精神力可恢复SAN值
- SAN值阶段效果：
  - 100-70：正常描述（"你面前是一条阴暗的通道"）
  - 70-40：轻微异常（"通道的墙壁似乎在微微蠕动"）
  - 40-10：严重幻觉（"血肉构成的食道在吞噬你的队伍"），怪物属性增强
  - 10-0：濒临崩溃，随机队员可能失控，降至0强制撤退

## 5. 战斗系统

### 5.1 基础模式

- **半即时回合制（ATB）**：角色速度属性决定行动条充能速度
- **战术预设（Gambit）**：战前设置AI逻辑，战斗中自动执行
  - 示例：盾卫在"队友血量<30%"时释放能量力场（嘲讽）
  - 示例：黑客始终攻击"护甲最低"目标
- 玩家可随时手动干预覆盖AI决策

### 5.2 职业体系

**女武神（物理系）：**

| 分支 | 定位 | 特点 |
| --- | --- | --- |
| 突击手 | 高输出 | 高暴击、高伤害、低防御 |
| 盾卫 | 坦克 | 嘲讽、护盾、高生命 |
| 处刑人 | 持续伤害 | 流血、暴击特效 |

**异格者（变异男性，法系/特殊系）：**

| 分支 | 定位 | 特点 |
| --- | --- | --- |
| 瘟疫使者 | DoT伤害 | 持续毒素、范围debuff |
| 脑波术士 | 控制/减益 | 眩晕、沉默、降速 |
| 狂暴体 | 爆发伤害 | 极高伤害但易失控攻击队友 |

### 5.3 异化值机制

- 异格者释放强力技能 → 累积异化值
- 异化值达到临界点 → 角色失控变身，攻击力暴涨但无法控制
- 失控状态下会攻击任何目标（包括队友）
- 主角使用"精神安抚"技能可降低异化值
- 战斗结束后异化值缓慢衰减

### 5.4 主角在战斗中的角色

- 主角不直接输出，作为辅助/指挥存在
- 提供全队光环：精神抗性加成
- 主动技能：全队回血、净化异常状态、安抚异格者
- 主角倒下 = 精神链接断裂 = 全队SAN值暴跌

## 6. 角色养成

### 6.1 等级与星级

- **等级**：通过战斗获取经验提升，提高基础属性
- **星级（义体改造）**：消耗同名角色碎片或通用芯片提升，解锁被动技能槽

### 6.2 转职（意识上传）

- 角色达到一定等级后可转职
- 需要特定材料（纳米核心，从BOSS掉落）
- 转职后解锁新技能树分支，外观变化

### 6.3 装备系统

- 装备通过蓝图+材料在纳米锻造间锻造
- 装备带有随机词条（如吸血+10%、暴击+5%）
- 装备品质：白 < 绿 < 蓝 < 紫 < 橙
- 可重铸词条（消耗算力）

### 6.4 主角成长

- 主角等级决定可探索黑门的层级（探索执照）
- 技能树分3条路线：治愈、安抚、感知
- 每次使用能力基因链裂痕+1（剧情数值，影响结局）

## 7. 社交与商业化

### 7.1 弱社交

- **遗言系统**：团灭地点留下"数据残骸"，其他玩家可发现
  - "安葬"：获得少量资源
  - "入侵"：挑战镜像队伍，赢取战利品
- **世界频道**：纯文字聊天，末世连接感

### 7.2 付费设计

- 神经接入舱招募（抽卡）：S/SS级角色
- 月卡/通行证：每日生物电和算力
- 皮肤：角色特殊义体外观（无数值加成）
- 不卖数值，不设体力墙
